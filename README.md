# Rust-Playground
1. A Marco Polo game
2. An actix_web simple float calculator

## Install Rust 

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Check Rust version

`rustc --version`

### Create a new directory called marco

`cargo new marco`
